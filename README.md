# project
Guess the Word Game
Простая игра на угадывание слов на Python в стиле виселицы. Игрок должен отгадать случайное слово за ограниченное количество попыток.

1️⃣Возможности
Случайное слово выбирается из встроенного списка
Можно угадывать по одной букве или целое слово
У игрока есть 8 попыток. Не допускается повторный ввод уже использованной буквы

2️⃣Требования
Python версии 3.xE
Никаких дополнительных библиотек не требуется

3️⃣Установка и запуск
Скопируйте код в файл с расширением .py, например guess_word.py
Откройте терминал в папке с файлом
Запустите игру командой python guess_word.py

4️⃣Как играть
Компьютер загадывает случайное слово из доступных вариантов
На экране отображаются пустые слоты вместо букв
Вводите по одной букве для угадывания
Можно ввести целое слово для мгновенной победы или поражения
При ошибочном вводе теряется одна попытка
Победа наступает при угадывании всех букв
Поражение наступает после исчерпания всех попыток

5️⃣Структура кода
words — список возможных слов для загадывания
secret_word — слово, которое загадал компьютер
guessed_letters — история ввода игрока
attempts — максимальное количество попыток
